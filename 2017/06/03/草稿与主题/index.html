草稿与主题

2017-06-03 • 标签: 博客入门

草稿

可以通过以下命令来新建草稿，草稿内容将被保存在source/_drafts 文件夹。

可以通过 publish 命令将草稿移动到 source/_posts 文件夹。

主题

本网站目前采用hexo-next主题。
部署详情请见：http://theme-next.iissnan.com/getting-started.html

Cannot Get /tags/

本主题下直接安装后，tags标签打开显示Cannot Get ／tags/
解决方法如下：

步骤一

新建一个页面（page），命名为tags。命令如下：

$ hexo new page “tags”

步骤二

打开新建文件夹 /source/tags/index.md，编辑内容如下：

---
title: Tagcloud
date: 2017-06-04 12:15:00
type: “tags”
---

步骤三

若你是按照上述方法安装next，无需步骤三。否则：
在菜单中添加链接。编辑 主题配置文件 ，添加 tags 到 menu 中，如下:(注意冒号后有空格)

menu:
home: /
archives: /archives
tags: /tags
